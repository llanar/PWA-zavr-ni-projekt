# PWA-zavr-ni-projekt
Završni projekt iz kolegija PWA 

Prije pokretanja aplikacije:

1. Download whole [repository](https://github.com/llanar/PWA-zavr-ni-projekt/tree/main) and save it under xampp/htdocs. Start XAMPP (Apache and MySQL).
2. Run [setup.sql](https://github.com/llanar/PWA-zavr-ni-projekt/blob/main/setup.sql) from http://localhost/phpmyadmin/index.php?route=/server/sql.
3. Open http://localhost/PROJEKT/index.php
